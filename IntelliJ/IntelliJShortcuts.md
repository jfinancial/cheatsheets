## My Top IntelliJ Apple Shortcuts

| Category          | Shortcut     | Description            |
|-------------------|--------------|------------------------|
| Navigation        | ⌘ + ⌥ + ←    | Navigate back          |
|                   | ⌘ + ⌥ + →    | Navigate forward       |
| Searching         | Double Sh    | Search everywhere      |
|                   | ⌘ + Sh + O   | Open Files             |
|                   | ⌘ + O        | Open Classes           |
|                   | ⌘ + L        | Go to line number      |
|                   | ⌘ + E        | Recent files           |
|                   | ⌘ + U        | Go to superclass       |
|                   | Ctl + ⇧ + B  | Go to type declaration |
| Selection         | ⌥ + ↑        | Extend selection       |
|                   | ⌥ + ↓        | Reduce selection       |
| Refactoring       | ⌘ + ⌥ + M    | Extract method         |
|                   | ⌘ + ⌥ + V    | Extract variable       |
|                   | ⌘ + ⌥ + F    | Extract field          |
|                   | ⌘ + ⌥ + C    | Extract constant       |
|                   | ⌘ + ⌥ + P    | Extract parameter      |
|                   | ⌘ + ⌥ + N    | Inline                 |
|                   | ⌘ + F6       | Change signature       |
|                   | Sh + F6      | Rename                 |
|                   | F6           | Move                   |
|                   | F5           | Copy                   |
| Settings          | ⌘ + ,        | Open settings          |
| Project Structure | ⌘ + ;        | Open project structure |
| Find (textual)    | ⌘ + F        | Find                   |
| Find (textual)    | ⌘ + R        | Replace                |
