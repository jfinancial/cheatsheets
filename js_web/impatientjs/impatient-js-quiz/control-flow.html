Quizzes » Control flow statements

1. Syntax

Which of these are legal?

// 1
if (x === 0) {
  foo(x);
}

// 2
if (x === 0) {
  foo(x);
} else {
  bar(x);
}

// 3
if (x === 0) {
  foo(x);
} else if (x < 0) {
  bar(x);
}

// 4
if (x === 0) foo(x);

// 5
if (x === 0) foo(x);
else bar(x);

// 6
if (x === 0) foo(x);
else if (x < 0) bar(x);

2. break

Which of the following statements are legal?

{
  break; // 1
}

label1: {
  break label1; // 2
}

while (true) {
  break; // 3
}

label2: while (true) {
  break label2; // 4
}
